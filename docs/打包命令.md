## Windows 打包命令

### 1. 激活虚拟环境
```powershell
.\.venv\Scripts\Activate.ps1
```

### 2. 确保虚拟环境包含 Python DLL（首次打包必做）
```powershell
# 复制系统 Python DLL 到虚拟环境（根据你的 Python 版本调整路径）
Copy-Item "$env:LOCALAPPDATA\Programs\Python\Python313\python313.dll" -Destination ".venv\Scripts\" -Force
```

### 3. 运行打包命令
```powershell
pyinstaller --onedir --noconsole --name MobiusJ --icon mobiusj/assets/logo.ico --add-data "mobiusj/assets;assets" mobiusj/main.py
```

### 参数说明
- `--onedir`: 生成单文件夹模式（所有依赖在一个文件夹内）
- `--noconsole`: 不显示控制台窗口（GUI 应用）
- `--name MobiusJ`: 指定生成的可执行文件名称
- `--icon mobiusj/assets/logo.ico`: 设置应用程序图标
- `--add-data "mobiusj/assets;assets"`: 添加资源文件（Windows 下用分号 `;` 分隔）

### 4. 打包完成
打包完成后，可执行文件位于 `dist/MobiusJ/` 文件夹内，运行 `MobiusJ.exe` 即可启动程序。

### 注意事项
- 确保已安装 PyInstaller：`pip install pyinstaller`
- 打包前确保程序在开发环境中能正常运行
- 首次打包会生成 `.spec` 文件，后续可直接使用 `pyinstaller MobiusJ.spec` 打包
