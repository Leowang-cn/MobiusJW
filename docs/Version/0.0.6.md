#1 判断题
  默认模块类型为知识

#2.题目录入，创建标签逻辑调整
  输入内容检索标签，回车时如果无选中，则自动弹出创建标签弹层。自带输入名称，模块名称默认知识，TAB到最终保存后，刷题题目录入页、刷新数据且自动选中。

#3.支持题目录入粘贴图片和查看题目图片
- 设置校验时，增加对图片存储位置（item_img_path）的判断）的校验
- 题目输入支持保存图片
  - 重新布局，新增图片输入（展示）
    - 用frame-grid布局
      1)整个界面分为两个水平分布的frame,和位于这两个frame下方的一个frame
      - 第一个frame 名叫 type_frame,尺寸为600*600，紧贴界面左侧和上边缘，主要包含以下元素，每组一行
        - 题目：文本；题目ID输入框；粘贴按钮，整体在一行，居中
        - 模块选中区域，居中
        - 标签文本输入框；创建标签按钮；刷新按钮，居中
        - 动态表格
        - 题目难度：文本；题目难度下拉框；居中
      - 第二个frame 名叫 img_frame,尺寸为600*600，紧贴界面右侧和上边缘，主要包含一个元素
        - 一个可以支持将剪切板的图片粘贴进来的区域
      - 第三个stance_frame,尺寸为20*1200，紧贴界面最下边缘，里面只有一行，包含两个按钮
      - 保存按钮和清空按钮
- 题目管理界面支持查看题目图片
