重构题目管理弹层的筛选区域：新规则如下：
1.题目检索
1.1. 添加一个搜索框，支持输入题目ID，点击输入框后的【检索】按钮，进行搜索，将item_tag_relations表中符合条件的题目ID，在下方动态表格中展示其对应信息。
2.题目筛选
题目筛选区域由如下部分组成：
- 标签筛选区：由选中标签展示区和标签筛选组合组成。标签筛选组合为：“模块类型 - 模块名称 - 标签名称”进行组合下拉筛选，最终标签名称下拉框中选择的标签，按照“module_type - module_name - tag_name”的格式，展示在选中标签展示区中。双击展示区中的标签，意味取消选中。
- 其他筛选：题目难度（item_level）、题目用途（item_usage）筛选，默认为空。
- 【查询】按钮，点击后，按照如下规则，在动态表格中展示数据：
  - 在item_tag_relations表中，查询每个选中标签关联的所有题目ID，若存在一个题目ID和每个选中标签都有关联数据，则这个题目ID符合筛选条件。
  - 在符合筛选条件的题目ID中，按照题目难度（item_level）、题目用途（item_usage）过滤，筛选出最终符合条件的题目ID。
  将最终符合条件的题目ID，在item_tag_relations表中对应的所有数据展示在动态表格中。
