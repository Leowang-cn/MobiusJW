批量添加判断题
添加判断题弹层按照如下规则重构
- 弹层尺寸：800*400,由上自下有五个frame.
- 第一个frame，只有1列，宽度为500，有一个多行输入框
- 第二行frame，有三列，
  - 第一列为：“模块类型：”+模块类型下拉框，取所有模块类型（module_type）默认选中“知识”。
  - 第二列为：“模块：”+模块下拉框，根据第一列模块类型下拉框选中的值，取对应模块类型下的模块名称（module_name）作为下拉选项，默认为空。
  - 第三列为：“标签：”+标签下拉框，根据第二列模块下拉框选中的值，取对应模块下的标签名称（tag_name）作为下拉选项，默认为空。
- 第三个frame有一个名叫“解析”按钮。
- 第四个frame暂时为空
- 第五个frame有两个按钮，一个名为“保存”、一个名为“清空”
- 点击“解析”按钮后，若多行输入框是否为空，如果为空，则提示“请批量填写题干”；如果不为空，按照如下规则解析：
  - 多行输入框有几行内容（空行不算），就在第四个dframe中新增对应行数，每行结构均相同，如下：
    - 第一列为：“模块类型：”+模块类型下拉框，下拉框选中值同点击解析按钮时、第二行第一列的选中值。
    - 第二列为：“模块：”+模块下拉框，下拉框选中值同点击解析按钮时、第二行第二列的选中值。
    - 第三列为：“标签：”+标签下拉框，下拉框选中值同点击解析按钮时、第二行第三列的选中值。
    - 第四列为一个单行输入框，宽度为500，每行一个从多行输入框内解析出的题干，若题干文本开头为大写字母+“．”，例如“A．”，则自动删除这个开头，保留后边的内容。
    - 第五列是两个互斥的单选框，第一个单选框为“正确”，第二个单选框为“错误”，默认两个都不选中。
- 点击保存按钮时：
  - 校验第四个frame的各行，是否存在至少一行满足条件：第一至四列全不为空、且第五列有一个单选框选中。
  - 将满足条件的行，按照如下对应关系，在数据库中TF_question表和TF_tag_relations表新增数据：
    - 第四列对应：TF_question表的TF_question_text
    - 第五列对应：TF_question表的TF_question_ans
    - 第三列的tag_name对应的tag_id,对应的是TF_tag_relations表的tag_id
  - 保存成功后，重置第1、2、4frame。
- 点击清空按钮，不保存数据，重置第1、2、4frame。
